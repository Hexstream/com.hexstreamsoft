⚓ NFS hosting » Preamble » My website history

I've been using NearlyFreeSpeech to host this website (hexstreamsoft.com) since 2011.

At first, I didn't have much content, so I was content with updating the site through fairly manual means, sometimes even just sshing in, launching a remote emacs instance and making minor edits that way.

Obviously that was a really bad way to go about things, but fortunately in February 2013 I somehow figured out that NearlyFreeSpeech does have support for updating one's website via git pushes. This has been a revolution for me.

⚓ NFS hosting » Preamble » My motivations for writing this article

For some reason, NearlyFreeSpeech does not advertise their git support anywhere. I think that's a shame, since in my view it's one of the best ways to go about things. Hence, I've decided to write this guide to describe my setup, which I love so much and might appeal to others too.

(I dream of even better setups, but that's a discussion for another day.)

I'd also like to say that managing your own website is a lot of fun! I think there's too many people that settle for, say, a hosted blog (such as tumblr, wordpress or blogger) just because they overestimate the complexities of managing a "real" website. It really isn't that hard. I hope this article helps some people getting started by giving some good answers to important questions that might otherwise discourage them.
