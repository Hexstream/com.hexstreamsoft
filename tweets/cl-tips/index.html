Emacs/Slime (4) ⚓

Slime Tip: M-x slime-cheat-sheet shows a very useful list of commands to get started. Unfortunately this is a much under-advertised feature!

— Jean-PhilippeParadis (@HexstreamSoft) July 20, 2012

Emacs Tip: C-h C-h shows help commands. First C-h: tree of help commands. Last C-h: list keystrokes/commands for current partial command!

— Jean-PhilippeParadis (@HexstreamSoft) July 20, 2012

Slime tip: Messed up your CL image (ex: package conflicts)? Invoke M-x slime-restart-inferior-lisp or ",restart-inferior-lisp" in the REPL.

— Jean-PhilippeParadis (@HexstreamSoft) August 26, 2012

Slime tip: C-c ~ (slime-sync-package-and-default-directory) in a .lisp file to sync slime REPL package and default directory with that file.

— Jean-PhilippeParadis (@HexstreamSoft) September 4, 2012

CL newbs: Please avoid inventing your own reader macros until you've acquired proper CL experience/taste. Do learn about built-in ones.

CL newbs: REPL tests tell you about your particular CL impl, not the CL standard! Check CLHS to ensure portability before reporting "bugs".
