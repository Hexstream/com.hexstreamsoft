1.1 Quick Reference

Here are all the symbols exported by place-utils.

Type Symbol Description
Macro setf-expanderlet Introduce local setf-expanders.

Macro with-resolved-places Evaluate subforms once, then access repeatedly safely.


Modify Macro updatef Like setf, except supply update functions
(to be called with old values) instead of new values.

Modify Macro bulkf Flexible mass updating of places.
An update function receives the old values as arguments
and returns the values to write back as multiple values.

Modify Macro funcallf Store into a place the result of calling
an update function with the old value of the place,
possibly along with additional regular arguments.
Modify Macro applyf Same as funcallf except apply is used.


Accessor Modif cachef Compute the value of a place only when it's first read.
Accessor Modif oldf Make the storing form of a place return old value(s).
Accessor Modif readf Easily make a place work in non-place contexts.
Accessor Modif tracef Output debug info when a particular place is accessed.

In this documentation, the term “Modify Macro” is used in a more general sense than traditional modify macros that have restrictions such as accepting only one place and then only as the first argument.

An “Accessor Modif[ier]” is a place that:

  1. is implemented as a macro;
  2. takes at least one place as argument;
  3. returns the setf-expander of this (these) inner place(s) with some modifications.
