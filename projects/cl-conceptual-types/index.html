cl-conceptual-types is a trivial Common Lisp library that simply creates a package that exports symbols for a few conceptual types "missing" from the COMMON-LISP package. Library authors that use and export these symbols instead of creating their own will avoid unnecessary symbol conflicts (or name mangling) for themselves and their users. These symbols are useful targets for various kinds of dispatching.

The conceptual type symbols "missing" from the COMMON-LISP package are:

  • SLOT
  • ACCESSOR
  • MACRO
  • SYMBOL-MACRO
  • READER-MACRO
  • PPRINT-DISPATCH-TABLE

The source code for cl-conceptual-types is hosted on GitHub.

cl-conceptual-types is in the Public Domain.
